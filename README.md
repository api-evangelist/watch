# WatchThis is a repo for managing machine readable artifacts for the [Watch APIs](http://dev.wewatchthematch.com/index.php?r=site/index). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [Watch APIs](http://dev.wewatchthematch.com/index.php?r=site/index).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](http://dev.wewatchthematch.com/index.php?r=site/index)- [Developer](http://dev.wewatchthematch.com/index.php?r=site/index)- [Twitter](https://twitter.com/#!/wewatchthematch)The [APIs.json](https://github.com/api-evangelist/watch/blob/master/apis.json) index provides links to all the relevant links for the Watch APIs, including the OpenAPI and Postman if they are present.
